LLM — 大脑 07 / 20

LARGE LANGUAGE MODEL

Agent 的大脑

LLM 是 Agent 的核心推理引擎，负责理解用户意图并做出决策。

  • 理解自然语言输入
  • 生成推理链 (Chain of Thought)
  • 决策下一步行动
  • 生成结构化输出（JSON / 函数调用）
  • 常用：GPT-4、Claude、DeepSeek

03 / AGENT LOOP

AGENT LOOP
核心循环

感知 → 思考
→ 行动 → 观察

PERCEIVE → THINK → ACT → OBSERVE

Agent Loop 详解 13 / 20

循环机制

Agent 不断循环执行四个步骤，直到任务完成或达到最大迭代次数。

  • Perceive 感知 — 接收用户输入，整理上下文
  • Think 思考 — 调用 LLM，分析和推理
  • Act 行动 — 执行工具调用或生成回复
  • Observe 观察 — 收集结果，判断是否完成
